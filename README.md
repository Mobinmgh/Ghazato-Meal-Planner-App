## What is Ghazato?

Ghazato (غذاتو) is an AI-powered meal planning PWA built specifically for Iranian users. It generates personalized weekly meal plans using traditional Iranian dishes, based on the user's health goals, weight, and dietary preferences.

No MyFitnessPal. No generic Western recipes. Just real Iranian food — قرمه‌سبزی, کوکو سبزی, آش رشته, کباب کوبیده — with accurate calorie counts and full recipes.

---

## Features

- **AI Meal Planning** — Personalized 7-day meal plans generated by AI based on user profile
- **Iranian Food Database** — Plans built exclusively from traditional and modern Iranian dishes
- **Full Recipes** — Each meal includes ingredients, step-by-step instructions, nutrition data, and cooking tips
- **Smart Shopping List** — Auto-generated weekly shopping list from the meal plan
- **User Accounts** — Phone number OTP login, plans synced across devices via Supabase
- **Subscription Model** — ZarinPal payment integration for monthly and quarterly plans
- **PWA** — Installable on Android and iOS, works offline
- **RTL First** — Built from the ground up for Persian (Farsi) with full RTL support
- **Responsive** — Mobile app experience + full desktop dashboard

## Getting Started

### Prerequisites

- Node.js 20+
- npm or yarn
- Supabase account
- Arvan Cloud account (for AI API)
- Kavenegar account (for SMS OTP)
- ZarinPal account (for payments)

### Installation

```bash
git clone https://github.com/Mobinmgh/Ghazato-Meal-Planner-App.git
cd Ghazato-Meal-Planner-App
npm install
```

### Environment Variables

Create a `.env` file in the root:

```env
# Supabase
SUPABASE_URL=your_supabase_project_url
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_anon_key

# AI (Arvan Cloud)
ARVAN_API_KEY=your_arvan_api_key
ARVAN_API_ENDPOINT=https://api.arvancloud.ir/ai/v1/chat/completions
ARVAN_MODEL=your_model_name

# SMS (Kavenegar)
KAVENEGAR_API_KEY=your_kavenegar_api_key
KAVENEGAR_SENDER=your_sender_number

# Payments (ZarinPal)
ZARINPAL_MERCHANT_ID=your_merchant_id
ZARINPAL_SANDBOX=true

# App
APP_URL=http://localhost:3000
```

### Database Setup

Run this SQL in your Supabase SQL editor:

```sql
create table profiles (
  id uuid primary key default gen_random_uuid(),
  phone text unique not null,
  created_at timestamp with time zone default now()
);

create table plans (
  id uuid default gen_random_uuid() primary key,
  user_id uuid references profiles(id) on delete cascade,
  plan_data jsonb not null,
  profile_data jsonb not null,
  created_at timestamp with time zone default now()
);
```

### Run

```bash
npm run dev
```

App runs at `http://localhost:3000`

## How It Works

1. User lands on the marketing page
2. Signs up or logs in with their Iranian phone number (OTP via Kavenegar)
3. Completes a 6-step onboarding: goal, gender, age, weight, activity level, disliked foods
4. AI generates a personalized 7-day meal plan using Iranian dishes
5. Plan is saved to Supabase — accessible from any device
6. User can view full recipes, generate a shopping list, and subscribe for weekly new plans
7. Subscription processed via ZarinPal
